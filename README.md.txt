# Sistema de Administración de Citas

## Instalación y configuración

1. Clona este repositorio: `git clone https://github.com/mjbrayan23/Evidencia.git`.
2. Accede a la carpeta del proyecto: `cd Evidencia`.
3. Ejecuta el archivo JAR para iniciar la aplicación.

## Uso del programa

Esta aplicación permite administrar citas médicas, incluyendo funcionalidades para registrar doctores, pacientes y programar citas.
